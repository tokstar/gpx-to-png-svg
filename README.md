# GPIX file to image converter
this can create PNG or SVG from a GPX file
clone this repo, then run:
```
npm install
mkdir ./gpx-files
mkdir ./output-images
```
this install libray and create input/output directories

to create PNGs 
```
node ./generatePngs.mjs
```

to create SVGs 
```
node ./generateSVGs.mjs
```
